# StambhaX API Boilerplate

StambhaX API Boilerplate is a robust, clean, and extensible .NET 9 Web API foundation designed to accelerate the development of commercial and enterprise applications. It provides the essential infrastructure—from configuration to architecture—allowing developers to focus on business logic rather than plumbing.

## Features
- **.NET 9:** Built on the latest and greatest Microsoft technologies.
- **OpenAPI Integration:** Pre-configured with Swagger UI for easy endpoint testing.
- **Ready for Scale:** Designed to act as the core for various applications.

## Usage & Licensing
StambhaX operates under a **Dual Licensing** model to support both the open-source community and commercial enterprise development.

### 1. Open Source (Free)
You are free to use StambhaX for open-source projects under the **GPLv3 License**. Note that if you use this free version, **your entire application must also be open-sourced** under the GPLv3.

### 2. Commercial (Paid)
If you intend to build a closed-source, proprietary, or commercial application (where you do not want to share your source code with the public), you must purchase a **Commercial License**.
